本文详细介绍如何配置Redis服务，使其能够接受远程访问。首先，需要编辑redis.conf配置文件，注释掉bind127.0.0.1并设置protected-mode为no。然后，重启Redis服务。接下来，通过防火墙命令添加6379端口，使其永久开放，并重新加载防火墙规则。文章还提供了测试连接的方法以及如何移除已添加的端口。

redis服务安装好之后，默认只能本机访问，未开启守护线程服务，编辑redis.conf配置文件,我安装的版本是

1、配置文件
vim redis.conf

将 bind 127.0.0.1 注释掉

protected-node yes 设置成no

重启redis
systemctl restart redis

2、开启防火墙端口

查看防火墙状态 systemctl status firewalld

开启防火墙 systemctl start firewalld.service

关闭防火墙 systemctl stop firewalld.service

添加制定需要开放的****端口

firewall-cmd --add-port=6379/tcp --permanent

firewall-cmd --reload

查看端口是否开放

firewall-cmd --query-port=6379/tcp

移除6379****端口

firewall-cmd --permanent --remove-port=6379/tcp
