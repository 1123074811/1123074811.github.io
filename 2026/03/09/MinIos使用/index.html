MinIO 启动方法如下：

  1. 基本启动命令

  打开命令提示符或 PowerShell，进入 minio.exe 所在目录，运行：

  minio.exe server D:\minio-data

  D:\minio-data 是数据存储目录，可以改成你想要的路径。

  1. 启动后会显示

  • API 地址：http://127.0.0.1:9000

  • 控制台地址：http://127.0.0.1:9001（Web管理界面）

  • 默认用户名：minioadmin

  • 默认密码：minioadmin

  1. 自定义端口和凭据

  set MINIO_ROOT_USER=admin

  set MINIO_ROOT_PASSWORD=your_password

  minio.exe server D:\minio-data –console-address “:9001” –address “:9000”

  1. 访问控制台

  浏览器打开 http://localhost:9001，用上面的用户名密码登录即可进入 Web 管理界面。

  常见问题

  • 确保数据目录存在且有写入权限

  • 密码至少8位字符

  • 如果端口被占用，换一个端口即可
